# Helpdesk Ticketing System

A pixel-perfect UI implementation of a Helpdesk Dashboard based on Dribbble designs.

## Features
- 3-pane layout (Sidebar, Ticket List, Detail View).
- Toggle between Public Chat and Internal Notes.
- Real-time filtering of messages based on status.
- Fully responsive input bar.

## Tech Stack
- React.js (Functional Components & Hooks)
- CSS3 (Flexbox & Custom Properties)

## How to Run
1. Clone the repo: `git clone <your-url>`
2. Install dependencies: `npm install`
3. Start the app: `npm start`
